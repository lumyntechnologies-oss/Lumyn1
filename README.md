# Lumyn Technologies Landing Page

A high-end, professional single-page landing page for Lumyn Technologies.

## Quick Start

### Clone the Project
```bash
git clone https://github.com/lumyntech/Lumyn1.git
cd Lumyn1
```

### View the Website
Simply open `index.html` in your web browser:
```bash
# Or use a local server
npx serve .
```

## Editing the Page

### Update Company Info
- Company name, slogan, and description: Edit the Hero section in `index.html`
- Contact details: Edit the Contact section

### Update Projects
- Edit the Projects section in `index.html` to add your own portfolio items

### Update Community Section
- Edit the Community section to reflect your actual member count and links

### Customize Colors
Find and replace these values in the CSS:
- `--deep-obsidian: #0A0A0A` - Background color
- `--electric-blue: #00B4D8` - Accent color

## Tech Used
- HTML5
- Tailwind CSS (CDN)
- FontAwesome Icons
- Google Fonts (Inter)
